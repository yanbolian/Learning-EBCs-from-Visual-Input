# Learning-EBCs-from-Visual-Input
main.m implements V1-RSC model of Lian et al. 2023 J Neurosci paper: https://doi.org/10.1523/JNEUROSCI.1071-22.2023
plot_EBCs.m is used in main.m to plot figures.
